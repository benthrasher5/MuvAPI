My API

Usage:

./API/collection/

GET

Returns all objects in collection. If collection doesn't exist returns 404.


./API/collection/id/

GET

Returns the objects in collection with the corresponding object . If collection/item doesn't exist returns 404.

POST

Upserts body's object to collection. If no id field on object, one will be created. Returns upserted or new object, or 404 if collection doesn't exist


./headcount/muvId/

GET

Returns headcount for given muv id as a JSON object with "headcount" field


./login

POST

Body must contain username and password fields. If user and password match one in the db, returns a newly established valid session string and corresponding userId.


./logout

POST

Body must contain session string. Effectively logs out a user by deleting the corresponding session from the db.


./session

POST

Body must contain a "session" string. If a session with the corresponding id exists in the db, returns status 200, otherwise 400.
